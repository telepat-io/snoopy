# `reboot`

`reboot` is an alias group for startup behavior commands.

## Supported Commands

- `reboot enable`
- `reboot disable`
- `reboot status`

## Examples

```bash
snoopy reboot enable
snoopy reboot status
```

Use this group if you prefer the user-facing language of “run on reboot/login” over the `startup` terminology.
