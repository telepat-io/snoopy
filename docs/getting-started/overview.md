# Getting Started Overview

Snoopy helps teams monitor Reddit for high-intent conversations, qualify results with AI, and run repeatable scanning jobs on a schedule.

Use this section to get from zero to a reliable first workflow.

## What You Need

- Node.js 20+
- npm 10+
- OpenRouter API key

## Suggested Path

1. Install Snoopy and configure credentials: [Installation](installation.md)
2. Run your first job and validate output: [Quick Start](quickstart.md)
3. Learn command surface details: [CLI Reference](../reference/cli-reference.md)

## Core Concepts

- Jobs define your target audience and qualification criteria.
- Runs execute scans and persist analytics.
- Daemon mode executes enabled jobs on cron schedules.
- Results are stored locally in SQLite and exportable to CSV.
